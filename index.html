Exclusão de dados - Estante Virtual

O aplicativo Estante Virtual, desenvolvido por GC DevsApps, não coleta ou armazena dados pessoais em servidores externos. Todos os dados são armazenados exclusivamente no dispositivo do usuário.

Como excluir seus dados

Você pode excluir os dados do aplicativo das seguintes formas:

  • Excluindo manualmente os livros e imagens dentro do aplicativo
  • Desinstalando o aplicativo, o que apagará automaticamente todos os dados armazenados localmente

Tipos de dados excluídos

São excluídos todos os dados inseridos no aplicativo, como informações de livros, imagens de capa e anotações. Nenhum dado é mantido após a desinstalação.
